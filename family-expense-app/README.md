# 家庭支出管理系統 (Family Expense Management System)

這是一套以家庭為單位的財務管理 Web 應用程式，協助家庭成員共同記錄、分析及管理日常收入與支出，輕鬆達成家庭財務目標。目前為 v1.0 MVP 版本（前端 Mock Data 驅動），展示了現代化的 UI/UX 設計與核心操作流程。

## ✨ 核心功能

- **📊 Dashboard 總覽**：一目了然的本月收支、預算使用率，以及即時更新的圓餅圖與最新交易。
- **📝 交易記錄**：完整記錄每一筆支出與收入，支援關鍵字搜尋過濾。
- **💰 預算管理**：可針對個別分類設定每月預算，透過進度條與顏色（Safe/Warn/Danger）監控使用狀況。
- **📈 統計分析**：
  - **分類統計**：圓餅圖與列表呈現各分類消費佔比。
  - **成員統計**：長條圖 (Bar Chart) 排行榜，輕鬆看出誰是本月消費王。
- **⚙️ 設定與管理**：自由新增、編輯、刪除「交易分類」與「家庭成員」，並為他們挑選專屬的代表色。

## 🛠️ 技術棧 (Tech Stack)

- **前端框架**: React 18 + Vite (TypeScript)
- **樣式系統**: Tailwind CSS 3 
- **狀態管理**: Zustand
- **路由管理**: React Router DOM 6
- **圖表繪製**: Recharts
- **Icon 庫**: Lucide React

## 🚀 快速開始 (安裝與啟動)

### 1. 系統需求
請確保您的電腦已安裝 [Node.js](https://nodejs.org/) (建議版本 18 以上)。

### 2. 安裝依賴
開啟終端機 (Terminal)，進入專案目錄並執行安裝指令：

```bash
cd family-expense-app
npm install
```

### 3. 啟動開發伺服器
安裝完成後，執行以下指令啟動本地端伺服器：

```bash
npm run dev
```

接著打開瀏覽器並前往 `http://localhost:5173` 即可開始體驗。

## 📖 操作指南

1. **開始記帳**：
   - 點擊左側選單的「交易記錄」，或是直接點擊右上角的「+ 新增交易」。
   - 輸入金額、名稱、日期，並選擇分類與成員，點擊儲存後資料會立刻更新到全系統。
2. **管理分類與成員**：
   - 前往「設定與管理」頁面，可以點擊「+ 新增分類」或「+ 新增成員」。
   - 點擊現有分類/成員旁的「編輯」按鈕可修改名稱與顏色，這些顏色會自動反映在圖表與列表上。
   - 若要刪除分類或成員，系統會自動檢查是否已綁定交易記錄，避免資料異常。
3. **查看統計與預算**：
   - 預算是以「月」為單位，進入「預算管理」可設定各分類的金額上限。
   - 到「統計報表」可以查看即時生成的圖表分析。
